# CryptoSentAI

CryptoSentAI is an AI-powered crypto intelligence platform that analyzes social sentiment and market data to forecast cryptocurrency price trends using machine learning and deep learning.

Author: Lethabo Mafihle James Moshabane
Location: South Africa

---

## Project Overview

CryptoSentAI integrates multiple data sources to study the relationship between social sentiment and cryptocurrency market movements.

The system collects signals from:

* Telegram crypto channels
* Twitter/X discussions
* Binance market price data

These signals are processed using NLP models and combined with machine learning and deep learning models to generate market insights.

---

## Key Features

### Multi-Source Data Pipeline

Collects data from:

* Telegram crypto signal channels
* Twitter/X crypto discussions
* Binance cryptocurrency market data

### Sentiment Analysis

Uses transformer-based NLP models:

* DistilBERT
* FinBERT

This dual-model approach helps reduce sentiment bias.

### Machine Learning Models

Implements multiple predictive models:

* Linear Regression
* Random Forest
* LSTM Deep Learning (time-series forecasting)

Predictions are combined using ensemble techniques.

### Trading Signal Engine

Generates signals based on predicted price movement and market sentiment:

* BUY
* SELL
* HOLD

### Backtesting Engine

Evaluates trading strategies against historical market data.

### Portfolio Simulation

Simulates investment performance using model-generated trading signals.

### Explainable AI

Uses SHAP analysis to identify which features influence model predictions.

### Visualization Dashboard

Interactive analytics dashboard built using Streamlit.
